## Instrucciones

 - Instalar los paquetes necesarios.
	 - `npm install`
 - Ejecutar el servidor json-server para generar una API.
	 - `npm  run  json:server`
 - Ejecutar el cliente WEB de ReactJS.
	 - `npm start`
